# 🗺️ Week 1 Roadmap — DSA Bootcamp 2026
*WNCC Learner Space*

---

Week 1 has **5 topics** and **15 problems** (~2/day). Here's your path through it 👇

---

## 🛣️ The Path

```
🟢 START
  │
  ▼
📝 Syntax
  │  Get comfortable with loops, conditionals, functions & basic I/O.
  │  Just write small programs and break things. Don't overthink.
  │
  ▼
⏱️ Time & Space Complexity
  │  Learn Big O before anything else. You should be able to look at
  │  a loop and say "that's O(n)" without thinking too hard.
  │  Everything from Week 2 assumes you know this.
  │
  ▼
📦 Arrays
  │  The most important topic this week. Start with traversal,
  │  then reverse, max/min, rotate. Once comfy → prefix sums
  │  and two-pointer.
  │
  ▼
🔢 Sorting
  │  Go in order:
  │  Bubble → Selection → Insertion → Merge → Quick
  │  First three? Code from memory.
  │  Merge Sort? Trace on paper FIRST, then code.
  │
  ▼
🏁 END OF WEEK
```

---

## 🧪 Daily Problem Set (15 problems, ~2/day)

Do 2 problems every day alongside whatever topic you're on. **Don't pile them up for the weekend.** 🙅

For each problem —
- 🤔 Think for at least 15 mins before writing any code
- ✏️ Get *something* working first, even if it's slow
- ⚡ Then ask — can I do this faster?

> Stuck after 20 mins? Look at the **hint**, not the full solution.
> And if you did peek — re-solve it from scratch the next day. That second attempt is where you actually learn. 💡

---

## ✅ Before You Move to Week 2

- Can you write a loop without Googling the syntax?
- Can you look at simple code and tell its time complexity?
- Can you implement Bubble Sort from memory?
- Have you attempted all 15 problems?

If yes — you're good. See you in Week 2 🚀
